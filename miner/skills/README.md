# Miner Workflow Skills

These public skills break the miner improvement loop into small steps.

Use them from the repo root:

- `prepare-benchmark-context`
- `improve-artifact`
- `run-local-eval`

Recommended order:

1. `prepare-benchmark-context`
2. `improve-artifact`
3. `run-local-eval`

Repeat until the report quality is where you want it.
